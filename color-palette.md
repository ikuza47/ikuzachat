# IkuzaChat Color Palette

## Primary Colors

### Background Colors
- **Primary Background**: `#0a0a0a` (Almost pure black) - Used for main page background
- **Card Background**: `#121212` (Very dark gray) - Used for card containers and UI elements
- **Input Background**: `#1a1a1a` (Dark gray) - Used for input fields and form elements
- **Border Color**: `#262626` (Medium-dark gray) - Used for borders

### Text Colors
- **Primary Text**: `#f0f0f0` (Light gray/cream) - Main text color
- **Secondary Text**: `#a0a0a0` (Medium gray) - Secondary information and labels
- **Button Text**: `#0a0a0a` (Almost black) - Text on accent buttons

### Accent Colors
- **Accent Orange**: `#f9c7a0` (Light orange/beige) - Primary accent/highlight color
- **Accent Orange Hover**: `#fbd8c0` (Lighter orange/beige) - Hover state for accent elements

## Secondary Colors (from Tailwind CSS)

### Gray Scale
- **Gray 100**: Very light gray
- **Gray 200**: Light gray
- **Gray 300**: Medium-light gray
- **Gray 400**: Medium gray
- **Gray 500**: Medium-dark gray
- **Gray 600**: Dark gray
- **Gray 700**: Very dark gray
- **Gray 800**: Nearly black gray
- **Gray 900**: Very nearly black
- **Gray 950**: Almost pure black

### Indigo Colors
- **Indigo 300**: Light indigo
- **Indigo 400**: Medium indigo
- **Indigo 500**: Primary indigo (focus/hover states)
- **Indigo 600**: Dark indigo
- **Indigo 700**: Very dark indigo

### Other Colors
- **Red 500**: Used for error states or special notifications

## Usage Context

### Backgrounds
- Main page: `#0a0a0a`
- Cards/panels: `#121212`
- Input fields: `#1a1a1a`

### Text Elements
- Headings and primary content: `#f0f0f0`
- Secondary information: `#a0a0a0`
- Accent text and highlights: `#f9c7a0`

### Interactive Elements
- Hover states for gray backgrounds: `#f0f0f0` (text), `#1a1a1a` (background)
- Hover states for accent elements: `#fbd8c0`
- Focus rings: `#f9c7a0` and indigo colors
- Toggle switches: `#f9c7a0` when active

### Borders
- Standard borders: `#262626`
- Accent borders: `#f9c7a0` with 50% opacity for special elements

## Color System Notes
- The overall design uses a dark theme with a near-black background
- The accent orange color `#f9c7a0` serves as the primary highlight color
- Text maintains good contrast with the dark background using light grays
- The color scheme supports accessibility with appropriate contrast ratios
- The Tailwind CSS color system provides additional subtle color options for various states and elements
